# Weather App

- A simple and clean weather application built with HTML, CSS, and JavaScript.
 It fetches real-time weather data using the OpenWeatherMap API and displays temperature, humidity, wind speed, and weather conditions for any city.

 ## Live Demo - https://wasim2934.github.io/weather-app/

## Features
- Search any city to get live weather updates
- Temperature in Celsius
- Dynamic weather icons
- Humidity & Wind speed details
- Modern glass-card UI with a clean sky background
- Fast, responsive, beginner-friendly project

# Tech Stack
- HTML – structure
- CSS – UI design & animations
- JavaScript – API fetch and logic
- OpenWeatherMap API
